# thor-client README

Thor-client is a client for Thor ([link](https://github.com/cs-24-pt-10-01/thor)) which is a HotSpot detection tool.

This client provides a graphical interface for both requesting the profiling of a program, and viewing the results of the profiling.

## Features

- Can request profiling of a program
- Can see the energy usage of each function
- Can view the results in real time in the form of graphs
